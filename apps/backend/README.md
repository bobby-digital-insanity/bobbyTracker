# Bobby Tracker Backend

Simple Node.js Express API for Bobby Tracker.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Server will run on http://localhost:3000

## API Endpoints

- `GET /` - Root endpoint
- `GET /api/health` - Health check
- `GET /api/hello` - Hello world

## Environment Variables

Create a `.env` file:

```
PORT=3000
NODE_ENV=development
```
